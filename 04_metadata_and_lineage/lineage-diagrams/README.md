# Lineage Diagram Specification

This folder will contain future lineage diagrams that show:
- source systems
- transformation or movement layers
- governed data stores
- shared reporting assets
- downstream consumers
- control points and reconciliation checkpoints

Suggested future files:
- source-to-report-lineage.png
- critical-kpi-lineage.png
- access-sensitive-data-flow.png
- cross-system-reconciliation-lineage.png

Each lineage diagram should identify:
- asset names
- system boundaries
- critical control points
- manual intervention points
- downstream dependencies
